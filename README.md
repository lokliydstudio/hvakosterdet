# Hva koster det?

En statisk, responsiv nettside med fem norske hverdagskalkulatorer.

## Publisering

1. Bytt `https://dittdomene.no/` i `index.html`, `robots.txt` og `sitemap.xml`.
2. Oppdater kontaktinformasjon og personvernerklæring.
3. Last opp alle filene til rotmappen på webhotellet.
4. Koble nyhetsbrevskjemaet til ønsket e-posttjeneste.
5. Erstatt annonseplassene med annonsekode først etter at samtykkeløsning og personvern er på plass.

## Filer

- `index.html` – forside og kalkulatorer
- `styles.css` – design og mobiltilpasning
- `app.js` – beregninger og interaksjon
- `personvern.html` – enkel personvernmal
- `kontakt.html` – kontaktside
- `robots.txt` og `sitemap.xml` – grunnleggende SEO

## Viktig

Beregningene er estimater. Test og kvalitetssikre alle formler og standardverdier før kommersiell publisering.
